# My Dotfiles

## Getting Setup

First, check out this repo in your $HOME directory using git

```
git clone git@github.com/sean-brandenburg/dotfiles.git
```

## Installing Dependencies

```
cd dotfiles
brew bundle --file=./Brewfile
stow .
```
