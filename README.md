# My Dotfiles

## Installing Dependencies
```
brew install git
brew install stow

brew install neovim
brew install ripgrep
```

## Getting Setup
First, check out the dotfiles repo in your $HOME directory using git

```
$ git clone git@github.com/sean-brandenburg/dotfiles.git
$ cd dotfiles
```

then use GNU stow to create symlinks

```
$ stow .
```
